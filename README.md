# blog_miemiehoho
## 个人博客系统

基于 Springboot 开发的前后端分离的个人博客系统 + 管理后台

后端技术：

- 核心框架：Spring Boot
- 安全框架：Spring security
- Token 认证：jwt
- MySQL
- Redis
- RocketMQ

## 主要功能

- 登录、注册、登出
- 导航栏：文章分类、标签、文章归档（按年、月份）
- 首页：文章列表
    - 侧边栏：最热文章、最热标签、最新文章
- 文章详情页：评论
- 统一缓存处理、统一日志记录
- 图片上传

### 管理后台

- 权限系统
    - 登录认证
    - 权限认证
    - 权限管理
    - 用户权限配置
- 写文章
    - 图片上传
- 标签管理
- 文章类别管理
- 文章管理

## 数据库设计

- 用户表、文章表、文章内容表、文章标签、文章类别、文章评论
- 管理员表、管理员配置权限表、访问日志表、权限表
